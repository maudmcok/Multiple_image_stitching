# Multiple Image stitching to Paranomic image in Python


### Requirements : 

- Python 2.7
- Numpy >= 1.8 
- OpenCV 3.1.0 contrib
	
		pip install opencv-contrib-python

### Project Structure : 
	
		|_ code -|
		|		 |-- pano.py
		|		 |-- txtlists-|
		|		 			  |--files1.txt .... 
		|	
		|_ images - |
		|			|- img1.jpg
		|			|- abc.jpg 
		|			.... and so on ...
